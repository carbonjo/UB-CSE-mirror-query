At the end of the semester, student groups from several CSE capstone classes present the culmination of three months of effort, hard work, (metaphorical) blood, sweat (well... caffeine really), and tears (see above). 

In 2019, we added the Database Bake-Off—a competition in which student database developers race their homemade databases to see whose system can complete a standard set of SQL queries in the fastest time.

Participating Courses

The courses that participate in CSE Demo Day vary by semester, but we often see:

  • CSE 442/542: Software Engineering
  • CSE 462/562: Database Systems
  • CSE467/LIN 667: Computational Linguistics
  • CSE 510: Special Topics
  • CSE 601: Data Mining and Bioinformatics
  • CSE 611: MS Project Development
  • CSE 622: Advanced Computer Systems
  • CSE 662: Languages and Runtimes for Big Data
  • CSE 700: Independent Study
  • CSE 701-CSE 749: Seminars
